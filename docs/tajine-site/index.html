Niveau 2 : Causal Reasoning

Pourquoi ca se passe ?

Comprendre les causes profondes, pas juste decrire les correlations.

Sans raisonnement causal

"Les creations d'entreprises ont baisse de 15%"

Descriptif uniquement

Avec raisonnement causal

"La baisse des creations (-15%) s'explique par la hausse des taux BCE (+2 pts), reduisant l'acces au credit. Ce facteur est NATIONAL - la baisse locale est conforme a la moyenne regionale (-14%)."

Explicatif + Actionnable

Methode des "5 Pourquoi" Automatisee

Observation Creations -15%
↓ Pourquoi ?
Cause 1 Moins de financements bancaires
↓ Pourquoi ?
Cause 2 Taux d'interet +2 points
↓ Pourquoi ?
RACINE Politique BCE anti-inflation (hors controle local)

Classification des Causes

MACRO Taux BCE, inflation Non actionnable
SECTEUR Crise automobile Reconversion
LOCAL Foncier, fiscalite Actionnable

Exemple: Analyse Territoriale Complete

async def territorial_analysis(territory: str, sector: str):
    """Workflow orchestre par TAJINE."""

    # Phase 1: Collecte (agents existants)
    data_result = await data_agent.step(
        f"Collecte entreprises {sector} a {territory}"
    )
    veille_result = await veille_agent.step(
        f"Scan veille {sector} {territory}"
    )

    # Phase 2: Integration Knowledge Graph (NOUVEAU)
    kg_result = await kg_agent.integrate(
        data=data_result, signals=veille_result
    )

    # Phase 3: Detection signaux faibles (NOUVEAU)
    signals = await signal_detector.detect_weak_signals(
        texts=veille_result.news
    )

    # Phase 4: Prediction avec incertitude
    simulation = await simulation_agent.step(...)
    uncertainty = await uncertainty_agent.quantify(
        predictions=simulation.predictions
    )

    # Phase 5: Validation (NOUVEAU)
    validated = await validator_agent.validate(
        predictions=simulation,
        uncertainty=uncertainty
    )

    return validated
